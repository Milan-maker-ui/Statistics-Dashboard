# Statistics-Dashboard
A simple Python-based Statistics Dashboard that performs basic statistical analysis on sales data and visualizes the results using histograms and boxplots.

🚀 Features
•	Calculate statistical measures:
o	Mean,
o	Median,
o	Mode,
o	Standard Deviation,
•	Visualize data distribution using:
o	Histogram,
o	Boxplot,
•	Easy-to-understand implementation for beginners in Data Science and Statistics.

🛠️ Technologies Used:
•	Python 3
•	NumPy
•	Matplotlib
•	Statistics Module
