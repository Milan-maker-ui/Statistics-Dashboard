# Statistics-Dashboard
A simple Python-based Statistics Dashboard that performs basic statistical analysis on sales data and visualizes the results using histograms and boxplots.

## 🚀 Features

* Calculate statistical measures:

  * Mean
  * Median
  * Mode
  * Standard Deviation
* Visualize data distribution using:

  * Histogram
  * Boxplot
* Easy-to-understand implementation for beginners in Data Science and Statistics.

## 🛠️ Technologies Used

* Python 3
* NumPy
* Matplotlib
* Statistics Module

## 📈 Sample Dataset

```python
sales = [120, 135, 150, 145, 160, 155, 140,
         170, 175, 180, 190, 200, 210, 220, 225]
```

## 📊 Output

### Statistical Summary

```text
Sales Data: [120, 135, 150, 145, 160, 155, 140, 170, 175, 180, 190, 200, 210, 220, 225]

---- Statistical Summary ----
Mean: 170.33
Median: 170
Mode: 120
Standard Deviation: 32.83
```

### Visualizations

* Histogram showing sales distribution.
* Boxplot displaying spread and outliers.
